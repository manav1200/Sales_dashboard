**Sales KPI Analytics Dashboard**
An interactive sales analytics dashboard built with Python, Streamlit, and SQLite, using the Superstore retail dataset.

Overview
Loads the Superstore CSV dataset into a local SQLite database
Runs SQL-based analytics queries to extract KPIs and trends
Displays results in an interactive Streamlit web app with Plotly charts
Supports dynamic filtering by region, category, and year from the sidebar

Project Structure
app.py - Main Streamlit application; handles sidebar filters, KPI cards, and all chart visualizations
queries.py - All SQL analytics logic; contains five query functions used by the app
load_data.py - One-time script to parse and load Superstore.csv into sales.db
Superstore.csv - Source retail dataset with orders, customers, products, and regions
sales.db - SQLite database generated by load_data.py


Dashboard Features
KPI cards showing total revenue, total profit, total orders, and unique customers
Monthly revenue and profit trend line chart
Year-over-year sales and profit growth bar chart
Regional performance breakdown by product category
Top 15 sub-categories ranked by profitability and margin percentage
Customer segment analysis with pie chart and revenue summary table


Analytics Queries (queries.py)
yoy_growth() - Calculates year-over-year sales and profit growth percentage using window functions
regional_performance() - Sales, profit, order count, and margin by region and category with ranking
top_products() - Top 15 sub-categories by profit with margin percentage and dense rank
customer_segments() - Revenue, profit, and average revenue per customer by segment
sales_over_time(region, category) - Monthly sales and profit trend with optional region/category filter


Tech Stack
Python
Streamlit - web dashboard framework
Plotly Express - interactive charts
SQLite - local database
Pandas - data manipulation
